# P0 Batch 3 — i18n migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: none — this batch is a single long sequential session that edits ~17 screens + ARB files + smoke tests + docs. Parallelism on ARB is unsafe (single file, serialized writes). One implementer-subagent runs tasks 0–9 top-to-bottom.

**Goal:** Land `docs/superpowers/specs/2026-04-20-p0-batch3-i18n-design.md` — every UI-chrome Chinese string becomes an ARB-keyed lookup, English becomes the default locale, and the zh-CN bundle preserves all current copy.

**Tech stack:** Flutter 3.11.5 / `flutter_localizations` / `intl ^0.19` / native `flutter gen-l10n`.

**Spec:** `docs/superpowers/specs/2026-04-20-p0-batch3-i18n-design.md`

          localizationsDelegates: AppLocalizations.localizationsDelegates,
          home: const LoginScreen(),
