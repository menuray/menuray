# DESIGN.md — MenuRay 设计系统

> 这是一份 agent-friendly 的设计系统定义，用作 Stitch 等 AI 设计工具的全局上下文。
> 推荐用法：每次新 Stitch 会话开头粘贴这一份，让模型先理解品牌系统，再按 `stitch-prompts.md` 顺序丢具体屏幕。

---

## 一、品牌

- **名称：** MenuRay（英文，单词，CamelCase）
- **Slogan：** 拍一张照，5 分钟生成电子菜单
- **定位：** 商家拍纸质菜单照片，5 分钟生成可分享的电子菜单（链接 / 二维码 / PDF）。**纯展示与分发**，不做点餐、不做支付。
- **目标用户：**
  - 商家端：中小型餐厅老板 / 店长，35–55 岁，没空也不懂排版
  - 顾客端：到店扫码 或 朋友分享链接 看菜单的人，包含中外语言用户

---

## 二、平台

- **商家端：** 移动端原生（iOS / Android）
- **顾客端：** 响应式 H5（手机优先）

---

## 四、字体

- **中文：** 思源黑体 / OPPO Sans（现代无衬线，规整易读）
- **英文：** Inter / DM Sans
- **标题：** 字重 600–700
- **正文：** regular
- **价格：** 半重号 (medium-bold)

---

## 五、布局与圆角

- **基线网格：** 8px
- **卡片圆角：** 16px
- **按钮圆角：** 12px
- **输入框圆角：** 10px
- **小标签 / chip：** 999px (pill)

---

## 六、风格基调

- **商家端：** 现代简洁、清晰高效、专业可信、易操作 — 大触控目标、清晰层级
- **顾客端：** 餐饮温暖感、食物诱人、信息清晰 — 大图、温暖光感
- **整体气质：** 稳重 + 温暖 + 一点琥珀金的高级感

---

## 七、Logo

- 方形 App Icon
- 背景：墨绿 `#2F5D50`
- 主体：暖米白色的菜单矩形，居中略偏左，略微倾斜
- 菜单的右下角向上翻卷，卷起的内角呈现琥珀色 `#E0A969` 高光
- 整体边角微圆
- **无任何文字 / 字母**
- 在 32px 小尺寸下仍能识别"翻起一角"的剪影

---

## 八、品牌字标 Wordmark

- 永远写成 "**MenuRay**"（CamelCase，"Menu" + "Ray" 拼合）
- 排版可选：
  - 水平："MenuRay"
  - 双色："Menu" 用墨绿 + "Ray" 用琥珀色（呼应 logo 的翻起一角）
- 不要翻译成中文
- URL / package 场景全小写 `menuray`（域名 / handle / 包名）

---

## 九、品牌出现位置

- **商家端：** 启动页 / 登录页 / 我的页 顶部出现 Logo + "MenuRay" + slogan
- **顾客端：** 菜单页底部小字「由 MenuRay 提供」
